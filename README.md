=======

This is a PBKDF2 algorithm implemented in Ada, tested against RFC6070 test
vectors. Currently PBKDF2_HMAC_SHA1, PBKDF2_HMAC_SHA256 and PBKDF2_HMAC_SHA512
are implemented.
